# AI Web Scraper & Summarizer

This project is an AI-powered web content summarization pipeline built with Python and Azure OpenAI.

## Features
- Fetches web pages safely with HTTP headers
- Cleans and extracts meaningful text from HTML
- Validates and truncates large content
- Summarizes content using Azure OpenAI
- Handles errors and unreliable sources gracefully

## Tech Stack
- Python
- Requests
- BeautifulSoup
- Azure OpenAI
- python-dotenv

## Setup

1. Clone the repository
2. Create a virtual environment
3. Install dependencies:
